ORDEMFLIX

O OrdemFlix é uma página web estática desenvolvida com HTML e CSS que funciona como um catálogo organizado dos episódios da temporada "Desconjuração" do RPG Ordem Paranormal, criado por Cellbit. O projeto tem como objetivo principal fornecer uma maneira prática e visualmente atraente para os fãs acessarem os episódios diretamente do YouTube. Estrutura Técnica: HTML:Cabeçalho com o título "ORDEMFLIX" Seção principal contendo: Player incorporado do YouTube com o teaser da temporada Grade de episódios com thumbnails clicáveis Links diretos para cada episódio no YouTube CSS:Estilização temática com tons de vermelho Uso da fonte Girassol para títulos (mesma usada no título original da campanha)

Quente ou Frio – Caça ao Tesouro Digital

"Quente ou Frio" é um jogo web interativo desenvolvido com p5.js, HTML e CSS, onde o jogador deve encontrar um alvo escondido na tela usando apenas dicas visuais. Inspirado na brincadeira infantil clássica, o projeto transforma essa experiência em um desafio digital minimalista e envolvente. Objetivo: Encontrar o ponto escondido na tela o mais rápido possível, usando o feedback visual de cores que indicam proximidade ("roxo" = perto, "amarelo" = longe).
